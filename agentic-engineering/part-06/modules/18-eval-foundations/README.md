# Eval foundations

*(Coming soon)*

Evaluating non-deterministic systems is fundamentally different from evaluating deterministic code. This module covers what evals are for an agent: task-completion suites (does it accomplish the goal?), trajectory analysis (is the path it took reasonable?), and stochastic evaluation (averaging over N runs because the agent will give different answers each time). Test design for systems that don't behave the same twice.
