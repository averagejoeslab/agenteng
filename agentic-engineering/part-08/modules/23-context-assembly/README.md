# Context assembly

*(Coming soon)*

What goes *into* each LLM call — system prompt, recalled memories, recent turns, tool schemas, tool results — isn't fixed. This module covers the assembly strategy: which pieces, in what order, sized to what budget. Memory (Modules 11-13) gives raw material; eval (Module 19) measures whether your assembly works; cost (Module 20) constrains how much you can include. Context assembly is where they all come together.
