# Prompt design

*(Coming soon)*

The system prompt is scaffolding for the model's behavior — not just instructions but the structure inside which it reasons. This module covers prompt design as engineering: sections (role, constraints, conventions, examples), few-shot examples chosen for the task, and iterative refinement using eval scores from Module 19 as feedback. Small prompt changes have outsized behavioral effects; treat them like code changes.
