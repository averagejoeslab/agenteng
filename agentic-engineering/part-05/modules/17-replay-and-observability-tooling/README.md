# Replay and observability tooling

*(Coming soon)*

Traces from Module 16 enable two things: **replay** (reconstruct a run from its trace, useful for reproducing user-reported bugs) and **integration with observability platforms**. This module covers the trace consumer landscape — Phoenix, Helicone, Langfuse, OpenTelemetry — and shows how to plug the agent's emitted traces into one. The agent gains a debug story.
