# Structured tracing

*(Coming soon)*

You can't fix what you can't see. This module emits structured traces of every LLM call (request, response, tokens, latency), every tool call (name, input, output, duration, error), and every state transition. Correlation IDs across a turn so a multi-step task is one trace. The trace becomes the foundation for debugging, replay, and evaluation.
